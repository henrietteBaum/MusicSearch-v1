# MusicSearch v1

MusicSearch ist ein Lernprojekt. Wir zeigen, wie man ein Python-Skript erstellt, das nach einem, vom Benutzer eingegebenen, Suchbegriff in der Datenbank von Apple iTunes Musiktitel und Interpreten sucht.

#### Konkret zeigt Version 1.0 wie man 

- vom Benutzer einen Suchbegriff abfragt und speichert,
- dem Suchbegriff für eine Abfrage der Datenbank von Apple-iTunes einsetzt,
- dazu die erfoderliche API (Application Programming Interfade) einbindet
- das vom Server zurückgelieferte Ergebnis interpretiert und in ein für Python lesbares Format umwandelt
- das Ergebnis in Textform in einem Terminal ausgibt

Die **Versionen 1.1 bis 1.4** erklären, wie man die Funktionslogik mit einer grafischen Benutzeroberfläche verbindet. Dazu wird Tkinter verwendet.

**Version 1.5** trennt Programmlogik und Benutzeroberfläche.
